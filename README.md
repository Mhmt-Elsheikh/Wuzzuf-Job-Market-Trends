Wuzzuf Job Market Trends

Description: Analyzed over 25K job postings to uncover key trends in the tech labor market. The dashboard provides a deep dive into high-demand skills (SQL, Python, Cloud), remote work adoption, and salary competitiveness. It features interactive maps and trend lines to help job seekers and recruiters understand market dynamics and applicant competition.

Tech Stack: Power BI, Data Cleaning, DAX, Geographic Visualization.
